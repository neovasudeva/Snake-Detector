Welcome!


Welcome! SnakeDetector.ml is a simple machine learning model built with Detectron2 used for real time object detection of snakes. You can upload a photo of a snake to see if the model can detect it. Do not upload anything that you don't want the developer to see! You have the option to allow us to save submitted images by checking the box below. Using your images, we can train the model to be better. However, once you've checked the box and submitted an image, the image will be saved in our database and you will not be able to delete it. For current issues with the model and future updates, visit the About page.

To start, upload an image and wait a couple of seconds while the model tries to find snakes in the image. Each box corresponds to a snake the model found. If no box appears, it means the model did not find any snakes.
