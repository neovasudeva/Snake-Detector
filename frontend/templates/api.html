API

Current Version: 1.0

SnakeDetector.ml is proud to say that we do provide REST API endpoints to submit images and get the training data used to train the current version of the model. Our endpoints are listed as such:

  • /api/v1.0/inferences/<save> accepts POST requests of JPG images. Replace 'save' with either 'true' or 'false' where 'true' will persist the data into our database and 'false' will not.
  • /api/v1.0/gpu takes a GET request and will return JSON on whether inference is being performed on a GPU.
  • /api/v1.0/train takes a GET request and will return a zip file of training images and labels used to train the current version of the model.
  • /api/v1.0/train takes a GET request and will return a zip file of labeled and unlabeled images used to test the model.
